# 电赛小车 · 电控软件知识库

备战 **全国大学生电子设计竞赛（NUEDC）省赛 / TI 杯** —— 「小车 / 运动控制 / 循迹追踪」类赛题的技术知识库。
负责方向：**软件 / 电控**（主控固件、软硬件联调、PID 控制、与 K230 视觉通信）。

> 这是一个 **Obsidian Vault**。用 Obsidian「打开文件夹作为库」即可获得双链跳转、反向链接、标签筛选与关系图谱。从 [`00-总览MOC.md`](00-总览MOC.md) 开始读。

## 技术栈

- **主控（主力）**：TI MSPM0G3507（Cortex-M0+）
- **主控（练手/备用）**：STM32（F103 / F407 / G431，CubeMX+HAL）
- **视觉**：嘉楠 Kendryte K230（CanMV），UART 回传
- **控制**：位置式/增量式/串级/模糊 PID + 前馈 + 滤波
