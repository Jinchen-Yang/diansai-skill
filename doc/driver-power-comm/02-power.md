# 电源系统（电赛核心）

> 资料采集 · 执行/电源/通信线。电赛小车电源是"翻车重灾区"：电机干扰、舵机拖复位、稳压发热限流。
> 配套：仓库 `power-design` skill（确定性校验）、`tools/power_budget.py`。**本文给方法与选型，最终预算用工具校验、人工批准。**

## 1. 电池（一次能量源）

| 类型 | 单节电压 | 常见组合 | 容量 | 放电能力 | 适用 |
|---|---|---|---|---|---|
| **18650 锂电** | 3.7V 标称(满 4.2V) | 2S(7.4V)/3S(11.1V) 串联 | 2000~3500mAh | 中 | 主流小车 |
| **航模锂电(LiPo)** | 3.7V | **2S(7.4V)/3S(11.1V)** | 300~2200mAh | **高(20~50C)** | 大电流/大功率车 |
| 镍氢 AA | 1.2V | 5~6 节 | ~2000mAh | 中 | 入门 |

- 12V 电机 → 3S(11.1V，满 12.6V)；多数循迹车用 **2S/3S 航模锂电**(放电倍率高，扛得住电机堵转冲击)。
- 锂电**带保护板/留欠压保护**；现场电量下降会掉速，标定时注意电量基准一致。

## 2. 稳压：LDO vs DCDC（核心决策）

| 类型 | 代表器件 | 输出 | 效率 | 纹波/噪声 | 发热 | 何时用 |
|---|---|---|---|---|---|---|
| **LDO** | AMS1117-3.3 / -5.0、LM1117 | 固定/可调 | **低**(=Vout/Vin) | **小、干净** | **大**(压差×电流全转热) | 小电流(<300mA)、对噪声敏感(模拟/传感器/MCU 后级) |
| **DCDC 降压(Buck)** | MP1584、LM2596、TPS5430 | 可调 | **高(85~96%)** | 较大(开关纹波) | 小 | 大压差大电流降压(电池→5V/3.3V) |
| **DCDC 升压(Boost)** | MT3608 | 可调 | 高 | 较大 | 小 | 电压不够要升(如 3.7V→5V) |

**关键参数**：
- **AMS1117**：典型压差(dropout)约 **1.1V@1A**，最大 1A，但**发热严重**——`P = (Vin−Vout)×I`，如 5V→3.3V@0.5A 发热 0.85W，必须留覆铜散热，**别用它从 12V 直接降 3.3V**(会烧/限流)。
- **MP1584**：1.5MHz 开关、SOIC8、效率 92~96%、输入 ≤28V、最大 3A，板子最小巧；电赛降压首选之一。
- **LM2596**：150kHz、输入 ≤40V、3A，老牌稳定但体积大、纹波相对高、需电感大。
- **TPS5430**：宽输入工业级 Buck，质量好，适合大压差。
- **MT3608**：升压，最大 2A 输入侧，适合小升压。

**典型策略**：电池(7.4~12V) → 先用 **DCDC(MP1584/LM2596) 高效降到 5V** → 再用 **LDO(AMS1117-3.3) 从 5V 干净地降到 3.3V** 给 MCU/模拟。即"DCDC 干粗活、LDO 做后级净化"。

## 3. 多路隔离供电分轨（电赛必做）

电机/舵机是强干扰源，**绝不能和 MCU、传感器共一条轨**。分轨原则：

```
                     ┌─ 电机功率轨 (VM): 电池直供 → 驱动 IC VM  (大电流, 脏)
 电池 7.4~12V ───────┼─ 舵机轨:        DCDC → 5~6V/≥2A 专供舵机  (大冲击, 独立)
 (LiPo 2S/3S)        ├─ 5V 数字/逻辑:  DCDC → 5V → 编码器/OLED/驱动 VCC
                     └─ 3.3V 数字:     LDO  → 3.3V → MCU/IMU/K230 接口 (干净)
                     ★ 所有轨 GND 必须共地(单点汇接), 否则信号基准乱
```

- **分轨**：数字 3.3V、模拟/传感器、电机功率、舵机各一路；模拟轨可再经 LC/磁珠滤波。
- **共地**：所有 GND 单点汇接到电池负极附近，避免地环路；电机大电流回路与信号地分开走线、末端汇合。
- **去耦**：每个 IC 电源脚就近 0.1µF 陶瓷 + 整体 10~100µF 电解；驱动 VM、舵机轨加大电容(470~1000µF)吸冲击。
- 电机驱动输入端可加 LC 滤波 / 续流措施，抑制电机换向尖峰回灌 MCU。

## 4. 电流预算 —— 按电机堵转电流算（不是标称！）★

**最重要的一条**：电源/稳压/导线/保险都按**堵转(失速)电流**预留，不能按额定标称。电机起步、爬坡、被卡时瞬间就是堵转电流（标称的 5~10 倍）。

**估算方法**：
```
堵转电流 I_stall ≈ 电机额定电压 / 电机内阻(Ra)         // 手册没给就用此式
单轨峰值 = Σ(该轨每个电机 I_stall) + Σ(舵机堵转) + 逻辑电流
电源/DCDC 额定电流 ≥ 单轨峰值 × 1.3 (留 30% 余量)
导线线径、保险按峰值选
```

**示例**（2 电机循迹车，GA25-370 堵转 3.2A）：
- 电机轨峰值 ≈ 2 × 3.2A = **6.4A** → 电池放电能力 + 驱动(TB6612 峰值 3.2A/通道，刚好)需满足；若常堵转考虑换 AT8236/DRV8701。
- 舵机轨(1× MG996R 堵转 2.5A) → DCDC ≥ 2.5×1.3 ≈ **3.3A** 选 ≥3A 模块。
- 3.3V 数字(MCU+IMU+K230 接口 ~300mA) → AMS1117(1A) 够。

> 校验交给 `tools/power_budget.py`：列出每轨负载(按堵转)、稳压器额定、余量是否 ≥30%；不达标重选器件。
