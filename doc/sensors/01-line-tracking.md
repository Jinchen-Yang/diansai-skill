# 循迹传感器：灰度（光学）与电磁

> 横向纠偏的核心。两条技术路线：**光学灰度**（黑/白线赛道）与**电磁**（铺 20kHz 漆包线赛道）。先看题目赛道用哪种线。

---

## 一、光学灰度循迹

### 1. 工作原理
红外/白光发射 + 光敏接收，靠**反射光强**区分黑白：黑线吸光→反射弱，白底反射强。把反射强度读成 0/1（数字）或连续值（模拟）。

### 2. 数字量 vs 模拟量 —— 决定接 GPIO 还是 ADC

| 类型 | 输出 | 接主控 | 阈值在哪 | 优缺点 |
|---|---|---|---|---|
| **数字量** | 每路 0/1（板上 LM393 比较器+电位器定阈值） | **GPIO** | 板上电位器手调 | 接线简单、省算力；分辨率差、阈值要拧电位器、对光照敏感 |
| **模拟量** | 每路反射强度连续值 | **ADC** | MCU 软件里算 | 分辨率高、可软件归一化/标定；占 ADC 通道+算力 |
| **智能（I2C/UART）** | 模块内部直接算好的**中线位置/偏差** | I2C 或 UART | 模块内部 | 最省主控资源、直出偏差；依赖模块、贵 |

> 工程结论：**做精确循迹优先模拟量接 ADC**（或直接上 I2C/UART 智能灰度模块直出偏差）。数字量适合入门或只需粗略左右判断。

### 3. 几路怎么选 + 间距
- 路数越多分辨率越高：5 路入门、**8 路主流**、12/16 路高分辨率（多用智能模块）。
- 探头间距 8~12mm（与线宽匹配）；探头离地高度固定后**不能再变**，变了要重标。
- 一字排开横向布在车头前方；可加前瞻（探头往前伸）提前预判弯道。

### 4. 阈值标定 + 中线提取（模拟量）
模拟量必须先**逐路归一化**（消除各管离散性和光照差异），再用**加权质心法**求黑线中心：

```c
// 8 路模拟灰度，adc[i] 为 ADC 原始值；标定时分别采全黑/全白记录 vmax/vmin
float sum_w = 0, sum_x = 0;
for (int i = 0; i < 8; i++) {
    float x = (adc[i] - vmin[i]) / (vmax[i] - vmin[i]); // 归一化到 0~1
    if (x < 0.15f) x = 0;            // 阈值去噪
    sum_w += x * (i + 1);            // 位置加权
    sum_x += x;
}
float pos = (sum_x > 0.1f) ? (sum_w / sum_x) : last_pos; // 全丢线则保持上一帧
float err = pos - 4.5f;             // 8 路中心 = 4.5，喂转向 PID
```

> 标定流程：开机让车头扫过全黑、全白各采一遍记 `vmax/vmin`；**换赛道/换光照必须重标**——这是循迹稳定的第一前提。
> 全丢线（`sum_x` 过小）必须保持上一帧或进入"找线状态机"，否则中线跳变会甩车。

### 5. MSPM0 采集要点
MSPM0G3507 的 12-bit SAR ADC（17 通道）配 **多通道顺序重复转换 + DMA**，一次扫完全部灰度通道，避免逐路阻塞。配置参考立创 MSPM0 ADC 例程。

### 6. 库 / 参考
- 数字量：无需库，直接读 GPIO。
- 模拟量：自写 ADC+DMA + 上面的归一化/质心，无现成"灰度库"，照例程拼。
- 智能灰度（感为等）：厂家给 I2C/UART 读取例程，直接读中线寄存器。

## 二、电磁循迹

### 1. 原理与信号链
赛道中心铺 **20kHz（部分赛事 10kHz）正弦交流**的漆包线，电感线圈感生电动势：
**线圈 → LC 选频 → 运放放大 → 检波（整流平滑）→ ADC 采样 → 差比和求偏差**。

### 2. LC 选频
并联谐振 $f_0 = \dfrac{1}{2\pi\sqrt{LC}}$，实战 **L=10mH 工字电感 + C=6.8nF（CBB/独石，温漂小）** 并联在电感两端，$f_0\approx19.3$kHz 对准 20kHz，放大有用信号、抑制其它频率噪声。

### 3. 放大 + 检波
- 运放（LMV358 / TLV 轨到轨）同相放大，增益用 100k 电位器整定，把毫伏信号放到 ADC 满量程附近、**留余量防饱和**。
- 检波常用 **BAT54S 肖特基**（低压降 ~0.3V）峰值检波 + RC 平滑（R=10k、C=0.01µF，时间常数 ≫ 50µs 信号周期、≪ 控制周期），输出正比于幅值的直流给 ADC。

### 4. 电感布局
- **水平电感（横放）**：正下方感应最强 → 测左右偏差、判在线/出线，常用 1~2 对（内外）。
- **竖直电感（立放）**：对垂直磁场敏感 → 辅助判十字、环岛、出入弯。
- 常见"一字四电感"：外侧 V1/V4、内侧 V2/V3。

### 5. 差比和求偏差（比纯差值稳，自动归一）
$$ err = \frac{V_{left} - V_{right}}{V_{left} + V_{right}} $$

```c
// 四电感差比和；vmax 为各电感在该赛道实测最大值（换赛道必重标）
float L = adc[0]/vmax[0], R = adc[3]/vmax[3];  // 外侧归一化
float err = (L - R) / (L + R + 1e-3f);          // -1~1，正负即方向
// |err|>0.75 强打方向，0.5~0.75 弱打
```

### 6. 与 MSPM0 内置模拟外设结合（省外围）
MSPM0G3507 自带 **2× 零漂零交越运放(OPA) + 3× 高速比较器(COMP，含基准 DAC) + 12-bit 4Msps ADC**：
- **OPA 内置可编程增益**：可把外部运放放大级搬进片内，省一片外部运放（仍建议外部做 LC 选频 + 检波，片内 OPA 做末级放大/缓冲）。
- **COMP + 基准 DAC**：做"在线/出线"阈值判断或过零检测，省外部比较器。
- 多电感统一走 ADC 多通道顺序转换 + DMA 一次扫完。
- 这些都在 **SysConfig** 里图形化配置引脚与增益。

### 7. 库 / 参考
电磁部分无"库"，是模拟电路 + ADC 采样 + 差比和算法；TI 给 MSPM0 模拟外设的 SysConfig 配置与应用笔记。

## 三、本册陷阱速记
- **模拟量走 ADC 引脚、数字量走普通 GPIO**——别接错脚类型。
- 数字灰度阈值在板上电位器，模拟灰度阈值在软件，智能灰度阈值在模块——搞清在哪调。
- 离地高度、光照、铜线电流变化都会改变信号——**现场必重标 vmax/vmin**。
- 电感检波的 RC 时间常数要兼顾"平滑掉 20kHz 载波"和"跟得上控制周期"，太大响应慢、太小有纹波。
- 很多模块标 5V 供电，逻辑输出可能是 5V，接 3.3V MCU 的 GPIO 要确认电平兼容（见 03/04 末节）。
