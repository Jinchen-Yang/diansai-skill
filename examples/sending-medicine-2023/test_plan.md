# 现场标定 / 验收清单 —— 智能送药小车（test-checklist 走查产出）
来源: examples/sending-medicine-2023/problem.yaml（功能 F1~F6）· 复用 KB10 赛中 checklist

## A. 分系统自检（连线/上电后）
- [ ] 各电源轨电压：VM / 舵机5-6V / 逻辑3.3V / K230 5V（先量再插模块）
- [ ] 电机正反转 + PWM 调速线性（左右一致）
- [ ] 编码器双轮计数方向正确、无跳变
- [ ] MPU6050 WHO_AM_I 读到、姿态不漂
- [ ] K230 上电握手通（收到 0x04 ACK）
- [ ] 8 路灰度原始值随黑白变化

## B. 现场标定（光照/赛道相关，每换场重标）
- [ ] 灰度阈值现场重标（config / 主控阈值）→ 中线稳定（F1）
- [ ] K230 巡线 ROI/阈值/曝光现场标 → error 输出稳定（F1/F5）
- [ ] K230 门牌数字模型/模板现场试，连续 N 帧采信（F5）

## C. 控制整定
- [ ] 内环速度 PID（编码器）：阶跃不超调过大、稳态无静差
- [ ] 外环循迹偏差 P/D：直道不画龙、弯道不冲线（F3）
- [ ] 路口/病房事件：里程 + 灰度双确认，不误触（F1/F4）
- [ ] 多病房路径：顺序/较短路径逻辑跑对（F4）

## D. 联调
- [ ] UART 双向通；故意丢字节验重同步不卡死
- [ ] 视觉结果 → FSM 采信链路正确（送达/识别触发）
- [ ] 全程计时显示正确（F6）

## E. 验收
- [ ] 全流程：取药→巡线→到房→识别→送达→返航→停车，声光提示（F1/F2/F3）
- [ ] 停车精度：车体完全进区（题目判定）
- [ ] **跑 ≥5 次记成功率**
- [ ] **低电量复测**（电压降改变 PID 表现）
- [ ] 限时达标（F6）

> 这些是人在真车上的活；agent 只给清单与判据，整定靠现场。控制参数/阈值此前都是占位。
