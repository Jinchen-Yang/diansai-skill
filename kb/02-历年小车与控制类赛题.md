# 📜 历年小车与控制类赛题

> [!abstract] 本篇速览
> 2011→2024 逐题解析：送药小车、目标追踪、倒立摆、循迹车，命题规律与趋势

## 一、定位与读法

本节聚焦国赛（NUEDC，全国大学生电子设计竞赛）控制类与小车类赛题，**逐题经多源交叉核对了年份 / 题号 / 官方题名**（以官方 `nuedc.org/problems/` 索引、`nuedc-training.com.cn` 培训网赛题发布、各赛区教委通知为权威，CSDN/知乎复盘为旁证），纠正了若干常见以讹传讹。先明确两个易错点：

- **国赛逢单年举办**（2011 / 2013 / 2015 / 2017 / 2019 / 2021 / 2023 / 2025），**双数年是"赛区赛暨模拟电子系统设计专题赛（TI 杯）"**。2024 的"自动行驶小车"即出自 2024 TI 杯专题赛，因**强制 MSPM0**、影响极大且与本队主控直接相关，故一并纳入。
- **题号会随组别（本科组 / 高职高专组）变化**。同一年份本科组用 A–H、高职高专组用 I–K（或更靠后字母）；部分小车题**两组共用同一题号**（如 2024 H 题同时面向本科组与高职高专组）。下文标注组别。
- **控制类题目的命题"模板"**：通常含两个相对独立的子系统 + 闭环控制 + 声光提示 + 计时 + 误差阈值。

## 二、逐题梳理（核心要求 / 技术点 / 难点 / 主流解法）

### 2011 国赛 C 题 智能小车（双车交替超车与领跑）
- **核心要求**：甲、乙两辆小车同时从起跑线出发、同向行驶，在固定跑道（含直道与弯道）上**交替超车、轮流领跑**，全程互不碰撞、不冲出赛道；不得借助车外设备控制小车，不得在赛道上另加标记。
- **技术点**：红外 / 光电循迹（TCRT5000 阵列）、车间通信与前后车判别（无线 / 红外测距）、差速 PID 调速、编码器测程。
- **难点**：双车**实时博弈与防撞协同**、高速循迹下不冲出。
- **模块**：减速电机 + L298N / TB6612、循迹板、码盘、无线（NRF24L01）。
- **注**：同年 B 题为"基于自由摆的平板控制系统"、F 题为"帆板控制系统"，亦属控制类，可一并参考。

### 2013 国赛 C 题 简易旋转倒立摆及控制装置
- **核心要求**：电机驱动旋转臂带动摆杆（杆长 15±5cm，距转轴 20±5cm）。**基本要求**：(1) 从自然下垂 0° 摆起，快速达到 −60°~+60°；(2) 逐步增大摆动幅度直至**完成圆周运动**；(3) 将摆杆人工拉至约 **165°（接近倒立）释放后启动控制，保持倒立 ≥5s，且旋转臂转角不超过 ±90°**。**发挥部分**要求**自动摆起并进入倒立保持**。
- **技术点**：角度传感（绝对值编码器 / 电位器 / MPU6050）、**起摆能量注入 + 倒立 LQR/PID 切换控制**、电机电流环。
- **难点**：起摆到倒立的**控制律切换**、机械间隙与摩擦补偿。
- **模块**：直流有刷电机 + H 桥、增量 / 绝对编码器、姿态传感器。

### 2015 国赛 B 题 风力摆控制系统
- **核心要求**：60~70cm 细管上端经万向节固定，下挂 2~4 只直流风机构成风力摆，摆下激光笔静止时端点距地 ≤20cm。控制风机使激光笔在地面画出：(1) 长 ≥50cm 直线段往返；(2) 任意方向直线；(3) 摆动状态释放后**短时内由运动恢复到静止**；**发挥部分**画**指定半径圆**并保持、抗风干扰后复位。
- **技术点**：MPU6050 解算摆角 / 方位、**多路风机矢量合成推力**、PID（位置环 + 姿态环）。
- **难点**：欠驱动、惯性大、**方向解耦**、抗扰恢复。
- **模块**：MPU6050、2~4 路直流风机 + MOS 驱动、激光笔。
- **注**：同年高职高专组 **J 题"小球滚动控制系统"**也是滚球类控制题，是 2017 滚球题的"前身"。

### 2017 国赛 B 题 滚球控制系统（民间俗称"板球控制系统"）
- **核心要求**：**边长 65cm 的光滑正方形平板**上均布 9 个 ∅3cm 圆形区域（编号 1~9），小球**直径 ≤2.5cm**；通过**控制平板二维倾斜**完成定点停留（如区域 2 停留 ≥5s）、沿指定路径滚动、限时通过多个编号区等，并显示计时（秒）。
- **技术点**：**电阻屏 / 摄像头测球位**、二维舵机倾台、**串级 PID（位置环 + 速度环）**、卡尔曼滤波估速。
- **难点**：球在光滑板上**二阶积分特性 + 时滞**，极易振荡发散；定位噪声。
- **模块**：四线电阻触摸屏或 OpenMV、2 路大扭矩舵机、平台机构。
- **注**：**官方题名为"滚球控制系统"**，"板球控制系统"为民间俗称。同年高职高专组 **L 题为"自动泊车系统"**（超声阵列 + 舵机转向 + 泊车轨迹），属小车类。

### 2019 国赛 B 题 巡线机器人
- **核心要求**：沿地面**标线自主循迹**，路径含直角、锐角、断线、交叉路口等复杂拓扑，限时完成并精确停车；发挥部分含更复杂路径 / 速度要求。
- **技术点**：红外反射阵列或灰度传感、**断线记忆与路口决策**、差速 PID。
- **难点**：复杂拓扑下的**路径判别**、高速循迹精度。
- **同年 A 题"电动小车动态无线充电系统"**（行驶中无线供能）也属小车类，侧重无线能量传输 + 行驶控制。

### 2021 国赛 F 题 智能送药小车
- **场景**：模拟医院药房↔病房送药。走廊两侧墙体由**黑实线**表示，地面画**居中红实线**，并放置标识病房号（1~8）的黑色数字可移动纸张；药房与近端病房（1、2 号）位置固定，中部 / 远端病房号（3~8 号）测试时随机设定。
- **工作流程**：人工将小车置于药房，**手持数字纸张供小车识别目标病房号**，装载约 200g 药品；小车检测装载完成后自动启动，沿走廊**循迹 + 识别 + 寻径**送达指定病房，亮红灯等待卸载，人工卸载后熄红灯返航，回到药房亮绿灯。
- **核心要求（分档）**：**基本要求**为单车将药品送至**近端 / 中部病房并返回，去程与返程均 <20s**；**发挥 / 综合要求**为双车协同送至同一中部病房或不同远端病房。
- **技术点**：**K210 / OpenMV 数字识别 + 循迹**、路口计数、状态机寻径、红 / 黑双线处理。
- **难点**：**视觉数字识别可靠性**（光照、角度）、循迹与识别融合、到位精度与时间。
- **模块**：K210（MaixPy）/ OpenMV、灰度阵列、减速电机 + TB6612、OLED、指示灯。
- **与本队相关**：把 K210 换成 **K230（CanMV）** 即当下主流升级路线。

### 2023 国赛 E 题 运动目标控制与自动追踪系统
- **核心要求**：两支激光笔各装在**独立二维电控云台**上，对着白色屏幕 / 靶纸（含黑色边框区）。红色光斑模拟运动目标，绿色光斑自动追踪。
  - **基本要求**：复位后一键启动，绿色光斑**2s 内追上红色光斑**，两斑中心距 **≤3cm**，并发连续声光提示；红斑可在 1min 内沿屏幕黑色边框**走一圈**等。
  - **发挥部分**：红斑沿**任意贴在白纸上的黑色矩形框（线宽约 1.8cm）**运动，绿斑实时跟随；目标快速移动时进一步收紧误差、加快响应。
- **技术点**：**OpenMV / K210 双色色块识别**、二维舵机 / 步进云台 PID、坐标系标定（像素→角度）、红绿斑同帧分离。
- **难点**：**2s 收敛 + ≤3cm**、运动目标实时性、双激光互不干扰、云台抖动抑制。
- **模块**：OpenMV4 / K210、2 套二维云台、激光笔、单片机（STM32 / CW32 等）、蜂鸣器。
- **与本队相关**：**视觉回传 + 云台 PID 范式**，正是 K230 + MSPM0 的典型练习题。

### 2024 TI 杯 H 题 自动行驶小车（本科组 / 高职高专组共用，强制 MSPM0）
- **核心要求**：场地 ≥220cm×120cm，两条对称半圆弧（R=40cm，黑色，线宽约 1.8cm）顶点定义 A/B/C/D。**(1)** A→B 停车，≤15s；**(2)** A→B→(弧)C→D→(弧)A 一圈，≤30s；**(3)** A→C→(弧)B→D→(弧)A 一圈，≤40s；**(4)** 在 (3) 路径上多圈、比拼最短时间。每过一个点声光提示一次。
- **关键限制**：**主控强制 TI MSPM0 系列**且芯片需裸露可验；**禁用摄像头**（回归传统传感）；车体 ≤25×15×15cm，轮式（禁履带 / 麦轮）、只能前进不可后退。
- **技术点**：灰度 / 红外循迹、**编码器里程 + 陀螺仪（HWT101 / JY61P 等）航向**、无线段（无引导线段）按里程 + 角度走位、PID 速度 / 航向。
- **难点**：禁摄像头下**弧线高精度循迹 + 直线段惯性导航**、停点精度、过点过冲抑制。
- **模块**：**MSPM0G3507（80MHz）** + TB6612 + 编码电机 + 六轴 / 单轴陀螺仪 + 灰度阵列。
- **与本队相关**：**直接对标本队 MSPM0 主控**，是最佳起步训练题。

### 2025 国赛 E 题 简易自行瞄准装置（车载云台 + 视觉，本科组，强制 MSPM0）
- **核心要求**：装置 = **自动循迹小车 + 瞄准模块**。小车沿外沿 **100cm×100cm 正方形黑线（线宽 1.8±0.2cm）**逆时针循迹；**车载二维云台控制激光笔**，靶上光斑 **∅<0.55cm**。在**距 AB 边外 50cm** 竖立 **A4 幅面紫外感光靶纸**，以靶心为圆心、半径 **2/4/6/8/10cm** 画同心圆环；行驶中 / 到位后**击中靶心或指定环**并计环数。
- **关键限制**：**小车循迹与电机控制必须用 TI MSPM0**；视觉用 OpenMV / 类似平台。
- **技术点**：5 路灰度循迹、编码器速度闭环、**OpenMV 识别靶心 + 车体运动补偿**、二维（步进 / 舵机）云台解算。
- **难点**：**运动中瞄准**（车速与云台联动补偿）、激光落点精度、坐标标定。
- **模块**：**MSPM0G3507** + 双步进 / 舵机云台 + OpenMV + 编码器 + 灰度阵列 + 激光笔。
- **与本队相关**：**循迹（MSPM0）+ 视觉（可换 K230）+ 云台**三合一，几乎是本队配置的"标准答案题"。
- **同年高职高专组 K 题"自动避障小车"**：超声 / 红外避障 + 循迹，难度较低，可作入门。

## 四、控制类共性公式与代码骨架（可直接用）

**位置式 PID（循迹 / 云台 / 球位通用）**：

$$u(k)=K_p e(k)+K_i\sum_{j=0}^{k}e(j)T_s+K_d\frac{e(k)-e(k-1)}{T_s}$$

循迹偏差 $e$ 由灰度阵列加权得到（以 7 路为例）：

```c
// 灰度阵列加权求偏差（-3..+3），再喂给位置式PID
int weight[7] = {-3,-2,-1,0,1,2,3};
int err = 0, active = 0;
for (int i = 0; i < 7; i++) { if (gray[i]) { err += weight[i]; active++; } }
err = active ? err / active : last_err;   // 全丢线则记忆上次
last_err = err;
int turn = pid_calc(&line_pid, 0, err);   // 目标偏差=0
motor_set(BASE - turn, BASE + turn);      // 差速
```

**云台像素→舵机角（视觉追踪，对标 2023E / 2025E）**：

```python
# OpenMV/CanMV: 找色块中心，转为云台增量PID（P+小D即可2s收敛）
err_x = blob.cx() - 160     # 图像中心160(320宽)
err_y = blob.cy() - 120
pan  += int(0.06*err_x)     # Kp_pan
tilt -= int(0.06*err_y)
uart.write(b'\xAA%c%c\x55' % (pan & 0xFF, tilt & 0xFF))  # 帧头AA 帧尾55
```

**K230 / MSPM0 串口约定建议**：固定帧 `帧头(0xAA) + 数据 + 校验(和校验) + 帧尾(0x55)`，MSPM0 端用 **DMA + 串口空闲中断**收包，避免丢帧。

## 五、命题规律 / 常考能力点 / 近年趋势

**命题规律（强模板化）**：
1. **"双子系统 + 闭环 + 声光 + 计时 + 误差阈值"**几乎每题必备（如 ≤3cm、≤2s、≤15s 等硬指标）。
2. 基本要求重"做到"，发挥部分重"做快 / 做准 / 抗扰 / 视觉升级"——**最后 1~2 档发挥分往往决定省一 / 国奖**。
3. 近年**主控被点名**：2024H、2025E 强制 **TI MSPM0**；摄像头是否禁用反复横跳（2024H 禁、2025E 允许配视觉）。

**常考能力点**：① PID / 串级 PID 整定；② 多传感器融合（编码器 + IMU + 灰度 / 视觉）；③ 状态机与路径规划；④ 单片机外设（定时器 PWM、编码器接口、UART-DMA、ADC）；⑤ 机电联调与机械精度。

**近年趋势——视觉 + 控制融合**：
- 2021F（数字识别）、2023E（双色斑追踪）、2025E（运动中瞄准）一路递进，**"视觉感知 + 运动控制闭环"已成控制类主线**。
- 主控生态从 51 / STM32 → **TI MSPM0**（赛事强推、LP-MSPM0G3507 价低、80MHz、内置编码器接口 / 比较器 / 12bit ADC）。视觉从 OpenMV / K210 → **K230（CanMV，算力更强、支持更复杂检测）**。
- **本队 MSPM0（电控循迹）+ K230（视觉回传坐标）+ STM32（备份）** 的配置，正好命中 2023E / 2024H / 2025E 的训练闭环。建议按 **2024H（纯电控打底）→ 2023E（视觉云台 PID）→ 2025E（车载视觉 + 云台综合）** 顺序刷题。

- 主要修正一处套题归属错误：原'2022省赛 跟随行驶系统/自动泊车电动车'同题号并列有误——2022年10月TI杯省级联赛实为A题'无线充电可循迹电动小车'+B题'具有自动泊车功能的电动车'，'跟随行驶'属另一套/赛区题，已分列澄清。
- 删除/降权臆造内容：'2016 无线充电电动小车'作为统一国赛题缺乏权威出处已删除，无线充电小车改以2019A题与2022A题为准。
- 补充遗漏赛题：2015高职J题'小球滚动控制系统'（2017滚球题前身）、2017高职L题'自动泊车系统'、2011 B/F题等控制类题目，增强了序列完整性。
- 组别/赛事性质澄清：2024H题出自'赛区赛暨模拟电子系统设计专题赛(TI杯)'且本科组与高职高专组共用题号；2025E题为本科组、K题为高职高专组；2024H禁摄像头而2025E允许配视觉，体现'主控点名+摄像头反复横跳'的近年命题趋势。
- 对本队最具训练价值的刷题路线得到验证：2024H(纯电控/MSPM0打底)→2023E(视觉云台PID)→2025E(车载视觉+云台综合)，恰好对应MSPM0循迹+K230视觉+云台的本队配置。
