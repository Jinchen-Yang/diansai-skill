# ⚙️ 电机驱动与执行机构

> [!abstract] 本篇速览
> 编码器电机、TB6612/DRV8701、PWM调速、麦轮运动学、测速

## 电机类型与选型

电赛小车（追踪/循迹/运动控制）的执行机构核心是直流有刷减速电机 + 霍尔编码器。下面按"是否带反馈"和"功率/响应"两个维度对比常见型号。

### 直流减速电机（带霍尔编码器，主力方案）

| 型号 | 机座 | 常见减速比 | 额定电压 | 减速后空载转速 | 堵转扭矩 | 编码器(电机轴) | 适用场景 |
|---|---|---|---|---|---|---|---|
| N20 | φ12mm | 1:30~1:298 | 6/12V | 100~600 rpm | 小 | 7线/13线霍尔 | 微型小车、机械臂关节 |
| GA25-370 / JGA25-370 | φ25mm | 1:21~1:34 | 12V | ~300~500 rpm | ~14.5 kg·cm | 11线/13线霍尔 | 中小车（最常用） |
| GM37 / GA37 | φ37mm | 1:10~1:90 | 12/24V | 中 | 中 | 11线/13线 | 中大车、负载大 |
| JGB37-520 | φ37mm | 1:30/1:20 | 12V | ~330/500 rpm | 大 | 11线霍尔 | 大扭矩越野/爬坡 |
| TT马达 | 塑料 | 1:48/1:120 | 3~6V | 低 | 多数无编码器 | — | 入门、无闭环要求 |

选型经验（多源交叉验证）：370/JGB37 系列**速度与扭矩最均衡**、N20 灵活但扭矩小、TT 马达便宜但无编码器不利于闭环。**追踪类赛题强烈建议选带霍尔编码器的 GA25-370 或 JGB37-520**，便于做速度闭环（PID）。GA25-370 典型参数：额定 12V/0.36A，堵转电流约 3.2A，减速后约 366 rpm。

### 其他电机类型

- **空心杯电机**：无铁芯，效率高（70%~90%），机械时间常数 <28ms（部分 <10ms），响应极快、无齿槽，适合云台/精密快速场合；但价格高、堵转易烧，电赛小车很少用。
- **步进电机（42步进=NEMA17，机座 42×42mm）**：开环可精确定位，无需编码器即可走固定步数；但高速力矩骤降、需专用驱动（A4988/DRV8825/TMC2208），且体积重、加速慢，循迹小车一般不用，仅在"精确定位送物"类任务考虑。
- **舵机（转向/抓取）**：见下表。

### 舵机参数（SG90 / MG996R / 数字舵机）

| 型号 | 类型 | 工作电压 | 扭矩 | 转速 | 角度 | 控制 |
|---|---|---|---|---|---|---|
| SG90 | 模拟 | 4.8~6V | 1.8~2.2 kg·cm | ~0.1s/60° | 180° | PWM |
| MG996R | 模拟金属齿 | 4.8~7.2V | 13~15 kg·cm | 0.19~0.2s/60° | 180° | PWM |
| 数字舵机 | 数字 | 6~7.4V | 大、抖动小 | 快、响应稳 | 180/270° | PWM/总线 |

**统一PWM控制原理**：周期 20ms（频率 **50Hz**），脉宽 0.5ms→2.5ms 对应 0°→180°，中位 1.5ms。占空比换算：`占空比 = 脉宽/20ms`，即 0.5ms=2.5%，1.5ms=7.5%，2.5ms=12.5%。

```c
// MSPM0/STM32: 50Hz, 周期20000计数(1us分辨率), 角度0~180
uint32_t angle_to_ccr(float deg){
    float pulse_us = 500.0f + deg * (2000.0f/180.0f); // 500~2500us
    return (uint32_t)pulse_us; // 定时器ARR=20000, 1计数=1us
}
```

舵机电流大（MG996R 堵转 >1A），**必须独立供电**（5~6V/2A），共地，不可直接由 MCU 的 3.3V/5V LDO 供。

## 电机驱动芯片对比与选型

这是电控的关键决策点。下表按"额定电流"从小到大排列，电赛追踪小车（GA25/JGB37，单电机额定 <1A、堵转 2~3.5A）。

| 芯片 | 拓扑 | 电压范围 | 持续/峰值电流 | 外置MOS | PWM建议 | 封装 | 备注 |
|---|---|---|---|---|---|---|---|
| **TB6612FNG** | 双H桥 | VM 2.5~13.5V(典型≤10V), VCC 2.7~5.5V | 1.2A / 3.2A(单通道) | 否(集成) | 10~100kHz(常用~20kHz) | SSOP24 | 电赛最主流，1芯片驱2电机，外围简单 |
| **DRV8871** | 单H桥 | 6.5~45V | 3.6A峰值 | 否 | 100kHz | SOIC8 | 单电机大电压，PWM限流 |
| **AT8236** | 单H桥(国产) | 5.5~36V | 4A连续/6A峰值, Rds_on≈200mΩ | 否 | — | ESOP8 | 国产替代DRV8870, VREF限流 |
| **A4950/AS4950** | 单全桥 | 8~40V | 1.8A连续/3.5A峰值 | 否 | — | SOIC8 | VREF+ISEN电流限制 |
| **DRV8701E/P** | 单H桥**栅极驱动** | 5.9~45V | 由外置MOS决定(可>10A) | **是(4×NMOS)** | **≤100kHz** | VQFN24 | 智能车/大功率首选，需画MOS |
| **L298N** | 双H桥(BJT) | 5~46V | 2A/通道, 压降大(~2~3V)发热严重 | 否 | <40kHz | Multiwatt15 | 老旧、效率低、不推荐 |
| **BTS7960/BTN7971(IBT-2)** | 双半桥模块 | 6~27V | 43A | 集成 | <25kHz | 模块 | 超大功率(爬坡/大车) |

### 选型建议

- **追踪/循迹小车（GA25-370、JGB37）首选 `TB6612FNG`**：单芯片驱两电机、3.2A 峰值足够、几乎不发热、外围只需 2 个电容，是平衡车/循迹车标配。
- **若电机较大或要做电流闭环**：选 `AT8236`（国产便宜、4A）或 `DRV8701`（外置 MOS、智能车级别可靠）。
- **大功率/爬坡/越野（电流 >5A）**：用 `BTS7960` 模块。
- **避免 `L298N`**：BJT 压降大、发热、效率低，仅入门。

### DRV8701 接线要点（外置MOS方案）

引脚关键：`VM(1)` 电机电源（0.1µF+≥10µF 旁路）；`GH1/GH2(17/24)`、`GL1/GL2(19/22)` 驱动高/低侧 NMOS 栅极；`SH1/SH2(18/23)` 接高侧 MOS 源极+低侧 MOS 漏极；`SP/SN/SO(21/20/11)` 电流采样放大；`IDRIVE(12)` 设栅极驱动电流（6mA~150mA 拉/12.5mA~300mA 灌）。控制：**8701E 用 PH/EN**（方向/使能），**8701P 用 IN1/IN2(双PWM)**，PWM **不超过 100kHz**，支持 100% 占空比。

### TB6612FNG 接线（最常用）

```
MCU --PWMA--> PWMA   MCU --AIN1/AIN2--> 方向
MCU --PWMB--> PWMB   MCU --BIN1/BIN2--> 方向
MCU --IO----> STBY   (置1工作, 置0全停)
VM=电机电源(≤10V), VCC=3.3/5V逻辑, GND共地
AO1/AO2->电机A, BO1/BO2->电机B
```
真值表：`IN1=1,IN2=0` 正转；`0,1` 反转；`1,1` 刹车(短路制动)；`0,0` 滑行(惯性)。**PWMA/PWMB 不可悬空**。

## PWM 调速、死区与刹车/滑行

- **调速原理**：PWM 占空比 `D=t_on/T` 决定平均电压 `V_avg = D·V_motor`，转速近似正比于 `V_avg`。
- **PWM 频率**：TB6612 推荐 **15~20kHz**（>20kHz 超出人耳避免啸叫；过高则开关损耗、占空比分辨率下降）。MSPM0/STM32 常用 `定时器频率/(ARR+1)` 计算，例如 80MHz / 4000 = 20kHz。
- **死区(dead-time)**：H桥同侧上下管不能同时导通，否则直通短路烧管。集成芯片（TB6612/AT8236）内部已加死区；**DRV8701 外置 MOS 必须靠 MCU 高级定时器死区或芯片配置保证**。MSPM0 的 **TIMA**（高级定时器）原生支持死区插入。
- **刹车(brake) vs 滑行(coast)**：两端同时拉低（`IN1=IN2=0`）= 滑行(高阻、惯性滑动)；两端同时拉高（`IN1=IN2=1`）= 刹车(电机两端短路、能耗制动，停得快)。追踪急停用 brake，省电巡航用 coast。

## 轮系与底盘运动学

### 两轮差速（最常用）

设车体线速度 `v`、角速度 `ω`、轮距(两轮中心距) `L`：

```
v_L = v - ω·(L/2)    // 左轮线速度
v_R = v + ω·(L/2)    // 右轮线速度
反解: v = (v_L + v_R)/2,  ω = (v_R - v_L)/L
```
车轮角速度 `ω_wheel = v_wheel / r`（r 为轮半径）。结构简单、循迹首选，需第三点（万向轮/牛眼轮）支撑。

### 麦克纳姆轮/全向轮（omni）四驱

辊子 45°，可全向移动。设 `vx`(前后)、`vy`(横移)、`ω`(自转)，`k=(L+W)/2`（L=轴距半，W=轮距半之和的等效系数），逆解（O型排布、轮序左前/右前/左后/右后）：

```
v1 = vx - vy - k·ω
v2 = vx + vy + k·ω
v3 = vx + vy - k·ω
v4 = vx - vy + k·ω
```
（不同博客轮序/符号定义略有差异，需按实际安装方向标定）。omni 轮（3 轮 120° 或 4 轮 90°）原理类似但无 45° 辊子角，计算更直接。

### 重心与机械稳定性

电池/驱动板尽量低置、居中以降低重心；高速过弯防侧翻需 `重心高度 / 轮距` 越小越稳。麦轮底盘四轮**必须同时着地**（加悬挂或保证地面平整），否则掉速、跑偏。

## 测速方案（编码器 / 霍尔 / 码盘）

### 霍尔编码器线数→速度换算

JGB37/GA25 霍尔编码器单相每圈 **11 线**（部分 13 线），电机轴一圈 11 个脉冲（单相单沿）。STM32/MSPM0 编码器模式做 **4 倍频**（A/B 双相上下沿）。

**每输出轴一圈总脉冲数 PPR**：
```
PPR = 编码器线数 × 4(四倍频) × 减速比
例: 11线 × 4 × 30 = 1320 脉冲/圈
```

**转速换算（M法，定时计数）**：在采样周期 `T`(s) 内读到计数 `ΔN`：
```
转速 n(rpm) = (ΔN / PPR) × (60 / T)
线速度 v(m/s) = (ΔN / PPR) × (2π·r) / T
```
其中 `r` 为轮半径(m)。例：T=10ms、PPR=1320、ΔN=132 → 每秒10圈 → 600rpm。

### M法 / T法 / M-T法

- **M法（测频，定时数脉冲）**：`n = 60·M₁/(p·T)`，高速精度高、低速误差大。追踪小车常用，T 取 5~20ms。
- **T法（测周，定角测时）**：`n = 60·f₀/(p·M₂)`，f₀ 为高频时钟，低速精度高、高速差。
- **M/T法**：两者结合，全速域精度好，但实现复杂。**常规循迹用 M 法即可**。

### MSPM0G3507 外设配置（主控方案）

- 共 7 个定时器、最多 22 路 PWM：**TIMA0/TIMA1**（16位高级定时器，**带死区**，最多 12 路 PWM）+ TIMG0/6/7/8/12（通用）。
- **PWM**：边沿对齐，向下计数，常用 `驱动电机 PWM`。
- **QEI（正交编码器接口）**：PHA/PHB 映射到 CCP0/CCP1，可选 IDX 索引，直接硬件 4 倍频计数，**省 CPU**。每个电机配一个 TIMG-QEI 读编码器、一路 TIMA 出 PWM。

```c
// 伪代码: 速度环采样(10ms)
int32_t cnt = QEI_getDirection_Count(TIMG); // 硬件4倍频计数
float n_rpm = (float)cnt / 1320.0f * (60.0f/0.01f);
QEI_resetCounter();
float pid_out = PID_compute(target_rpm, n_rpm);
PWM_setDuty(TIMA, pid_out); // 速度闭环
```

**电控联调顺序建议**：先开环 PWM 验证正反转/接线 → 编码器读数验证(手转电机看计数增减) → 单电机速度环 PID → 双电机差速 → 加循迹/视觉(K230) 外环。K230 通过 UART 回传线偏差/目标坐标，MSPM0 做位置外环 + 速度内环串级 PID。

## 🔑 关键要点（速记）

- 追踪/循迹小车动力首选带霍尔编码器的 GA25-370 或 JGB37-520(12V)，配 TB6612FNG 驱动：单芯片驱两电机、峰值3.2A够用、几乎不发热、外围最简，是平衡车/循迹车事实标配。
- 驱动选型按电流分档：TB6612(1.2A/3.2A,集成)→AT8236(4A国产)/DRV8871(3.6A)→DRV8701(外置4×NMOS,可>10A,智能车级)→BTS7960(43A大功率)；坚决避开 L298N(BJT压降大、发热、效率低)。
- DRV8701 是唯一需要外置 MOS 的栅极驱动方案：GH/GL 驱动高低侧 NMOS、SH 接半桥中点、IDRIVE 设栅极电流、PWM≤100kHz；E型用PH/EN、P型用双PWM，可靠性高但要自己画MOS和死区。
- 编码器 PPR(输出轴每圈脉冲) = 线数 × 4(四倍频) × 减速比，典型 11×4×30=1320；速度 n(rpm)=(ΔN/PPR)×(60/T)，线速度 v=(ΔN/PPR)×2πr/T。
- 测速方法：M法(定时数脉冲)适合高速/常规循迹，T法(定角测时)适合低速，全速域用M/T法；常规追踪车用M法、采样周期5~20ms即可。
- PWM调速 V_avg=D·V_motor，频率建议 15~20kHz(避啸叫又不过度开关损耗)；IN1=IN2=1 为刹车(能耗制动,停得快)、IN1=IN2=0 为滑行(高阻惯性)。
- 舵机统一 50Hz/20ms 周期，脉宽 0.5~2.5ms 对应 0~180°(中位1.5ms)；MG996R 堵转>1A 必须独立5~6V供电并共地，SG90仅2kg·cm适合轻负载。
- 两轮差速：vL=v-ω·L/2, vR=v+ω·L/2(反解 v=(vL+vR)/2, ω=(vR-vL)/L)，结构最简、循迹首选；麦轮全向 v1~v4=vx∓vy∓k·ω，轮序符号需按实际安装标定且四轮必须同时着地。
- 主控 MSPM0G3507 外设匹配度高：TIMA0/1 高级定时器自带死区可出PWM，TIMG-QEI 硬件正交解码(PHA/PHB→CCP0/CCP1)直接4倍频计数省CPU，每电机一路TIMA出PWM+一路QEI读编码器。
- 电控联调顺序：开环验证接线/正反转 → 编码器读数(手转看计数) → 单电机速度环PID → 双电机差速 → K230经UART回传线偏差做位置外环，构成位置(外)+速度(内)串级PID。
