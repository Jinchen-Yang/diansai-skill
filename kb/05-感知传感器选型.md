# 🛰️ 感知传感器选型

> [!abstract] 本篇速览
> 灰度/电磁循迹、IMU 姿态解算、TOF 测距、里程计

## 总览：四类感知如何分工

电赛小车感知通常是「循迹（横向偏差）+ IMU（航向/姿态）+ 里程计（纵向位移）+ 测距（避障/定位）」的多源融合。本节给出每类的选型、电路、公式与代码骨架，主控以 **MSPM0G3507**（12-bit SAR ADC，17 路多路复用通道）为主，STM32F103/F4 为备，视觉偏差由 **K230(CanMV)** 串口回传。

> 核心分工：**横向纠偏靠循迹（灰度/电磁/视觉）**，**走直线与定角度转弯靠 IMU 航向角**，**定距离停车/走 N 米靠编码器里程计**，**避障/到边检测靠 TOF/超声波**。

---

## 一、光学循迹：TCRT5000 与多路灰度

### TCRT5000 红外对管

- 原理：红外发射管 + 光敏三极管，反射光强决定光敏管导通。黑线吸光→反射弱→输出高；白底反射强→输出低。`LM393` 比较器 + 多圈电位器调阈值，数字口输出，驱动电流 >15 mA。
- 探测距离 **1–15 mm 可调**，对环境光、离地高度、赛道反光极敏感，必须现场重新标阈值。
- 数字模块只给 0/1，分辨率差；**做精确循迹要取模拟量**（直接接 ADC，读反射强度连续值）。

### 多路灰度模块（感为/GW 等）选型

| 模块 | 通道 | 输出 | 间距 | 适用 |
|---|---|---|---|---|
| TCRT5000 数字 ×N | 自拼 | 数字 0/1 | 自定 | 入门、低成本 |
| 感为 8 路模拟灰度 | 8 | 模拟/ADC | 8–12 mm | 工业/电赛主流 |
| 感为 12/16 路智能灰度 | 12–16 | **IIC/UART 直出位置** | 1–8 mm | 高分辨率、省 ADC |

智能灰度（IIC/UART）内部已做归一化与中线提取，直接回传「中线位置/偏差」，省主控 ADC 与算力，强烈推荐给软件主攻的队伍。

### 阈值 / 归一化 / 中线提取

每路先做归一化（消除各管离散性与光照差异）：

$$ x_i = \frac{v_i - v_{i,min}}{v_{i,max} - v_{i,min}} $$

中线（黑线中心）用**加权质心法**（位置 = 各路权重×位置的归一化和）：

$$ pos = \frac{\sum_{i=1}^{N} i \cdot x_i}{\sum_{i=1}^{N} x_i} $$

偏差 = `pos - (N+1)/2`，喂给转向 PID。代码骨架：

```c
// 8路灰度，gray[i]为ADC归一化后(0~1)，黑线值大
float sum_w = 0, sum_x = 0;
for (int i = 0; i < 8; i++) {
    float x = (adc[i] - vmin[i]) / (vmax[i] - vmin[i]); // 归一化
    if (x < 0.15f) x = 0;          // 阈值去噪
    sum_w += x * (i + 1);
    sum_x += x;
}
float pos = (sum_x > 0.1f) ? (sum_w / sum_x) : last_pos; // 全丢线保持上次
float err = pos - 4.5f;            // 8路中心=4.5
```

要点：标定时分别采黑/白记录 `vmax/vmin`；全丢线（`sum_x` 过小）要保持上一帧或进入找线状态机，否则中线跳变会让车甩出。

## 二、电磁循迹：20 kHz 参考线检测

赛道中心铺通 **20 kHz（部分赛事 10 kHz）正弦交流**的漆包线，电感线圈感生电动势，经选频→放大→检波→ADC。

### 1) LC 选频

并联谐振频率：

$$ f_0 = \frac{1}{2\pi\sqrt{LC}} $$

实战常用 **L=10 mH + C=6.8 nF**，代入得 $f_0 = 1/(2\pi\sqrt{10\text{m}\times6.8\text{n}}) \approx 19.3\ \text{kHz}$，正好对准 20 kHz。10 mH 工字电感 + 6.8 nF（CBB/独石电容，温漂小）并联在电感两端。谐振放大感应信号、抑制其它频率噪声。

### 2) 放大 + 检波

- 运放（如 `LMV358/TLV系列`轨到轨）做同相放大，增益用可调电阻（如 100 kΩ 电位器）整定，把毫伏级信号放到 ADC 满量程附近，**留余量防饱和**。
- 检波两种：
  - **二极管峰值检波**：常用肖特基 `BAT54S`（低压降 ~0.3 V，适合小信号）+ RC 平滑（如 R=10k、C=0.01µF，时间常数远大于 50 µs 信号周期、远小于控制周期），输出正比于幅值的直流给 ADC。
  - **有效值/精密检波**：运放 + 二极管的精密整流，精度高但元件多。电赛多用峰值检波够用。

### 3) 电感布局

- **水平电感（横放）**：磁感线与线圈面平行，正下方感应最强 → 用于左右偏差与「在线/出线」判断，常用 1 对或 2 对（内外）。
- **竖直电感（立放）**：对垂直磁场敏感 → 辅助判断十字、环岛、出入弯。
- 常见「一字四电感」或「八字 + 中间」组合：外侧 V1/V4、内侧 V2/V3。

### 4) 差比和（差和比）求偏差

比纯差值更稳，自动归一、抑制整体幅值波动（高度/电流变化）：

$$ err = \frac{V_{left} - V_{right}}{V_{left} + V_{right}} $$

多电感时分内外两组：外侧 `(V4-V1)/(V4+V1)`，内侧 `(V2-V3)/(V2+V3)`，再加权。归一化用各电感在该赛道的最大值：`value/max*100`（实测 `vmax≈2.89V` 量级，**换赛道必须重标**）。

```c
// 四电感差比和
float L = adc[0]/vmax[0], R = adc[3]/vmax[3]; // 外侧归一化
float err = (L - R) / (L + R + 1e-3f);         // -1~1，正负即方向
// 偏差区间映射控制强度：|err|>0.75 强打，0.5~0.75 弱打
```

### MSPM0 采集

MSPM0G3507 12-bit SAR ADC（17 通道），配 **多通道顺序重复转换 + DMA**，一次扫完所有电感/灰度通道，避免逐路阻塞。参考立创/CSDN 例程做 ADC+DMA 序列采样。

## 三、IMU 惯性测量：选型与姿态解算

### 芯片对比

| 型号 | 轴 | 接口 | 陀螺噪声 | 特点/价位 |
|---|---|---|---|---|
| MPU6050 | 6 | IIC(≤400k) | ~5 mdps/√Hz | 最便宜、资料最多、有 DMP |
| MPU9250 | 9 | IIC/SPI | 中 | 带磁力计，已停产难买 |
| ICM-20602 | 6 | IIC/SPI(10M) | ~4 mdps/√Hz | 智能车经典、低噪 |
| **ICM-42688-P** | 6 | IIC(1M)/**SPI 24M** | 最低 | 稳定性优于上述，量程±16/8/4/2g、±2000…dps，灵敏度高，**推荐主力** |
| JY61/JY901S(维特) | 6/9 | UART/IIC | 模块级 | **内置卡尔曼直出欧拉角**，静态 0.05°、动态 0.1°，9 轴含磁补航向 |

选型建议：
- **想省事、软件主攻**：直接上 **JY901S**，UART(默认9600，可到921600)直接读 roll/pitch/yaw，Z 轴航向用磁力计融合解决积分漂移，开箱即用，适合走直线/定角转弯。
- **想自己解算、追求高动态/高采样**：**ICM-42688-P**（SPI 24 MHz）或 ICM-20602，自写互补/Mahony。
- **预算极低/练手**：MPU6050（注意纯 6 轴 yaw 会缓慢漂移，转弯靠相对角度即可）。

### 姿态解算方法对比

| 方法 | 难度 | 频率/精度 | 说明 |
|---|---|---|---|
| 一阶/二阶互补滤波 | 低 | 高，调参即上线 | 加速度低通(长期准) + 陀螺高通(短期准)，roll/pitch 够用 |
| 卡尔曼滤波 | 中 | 高 | 单轴角度估计经典，计算量略大 |
| Mahony 互补 | 中 | 高、稳定 | PI 调 Kp/Ki，四元数输出，智能车常用 |
| Madgwick | 中 | 高 | 梯度下降，参数少 |
| MPU6050 DMP | 极低 | ≤200 Hz、精度一般 | 调库即用，速度慢 |

互补滤波最小实现（俯仰角）：

$$ \theta = \alpha(\theta_{prev} + \omega_g \cdot dt) + (1-\alpha)\theta_{acc},\quad \alpha \approx 0.98 $$

```c
// 一阶互补滤波 pitch
float acc_pitch = atan2f(ay, az) * 57.2958f;
pitch = 0.98f * (pitch + gyro_x * dt) + 0.02f * acc_pitch;
```

走直线常用做法：只用 **yaw 角速度积分 / 模块 yaw** 作航向反馈，构成「航向环 + 编码器速度环」双 PID。

## 四、测距：超声波 / TOF / 红外

| 传感器 | 量程 | 精度 | 接口 | 频率 | 备注 |
|---|---|---|---|---|---|
| HC-SR04 超声波 | 2cm–4m | ±1–3mm(理想) | Trig/Echo(GPIO) | ~20Hz | 最便宜，波束宽(~15°)、受温度/软表面影响，易误测 |
| VL53L0X 激光TOF | 最大2m | ±3% / <±3mm | IIC | <30ms | 不受目标颜色反射率影响，窄视场，精准 |
| VL53L1X 激光TOF | 最大4m | 高 | IIC | 可达50Hz | L0X升级，更远更快，可ROI |
| 红外避障 | cm级 | 二值 | GPIO | 快 | 只判有无障碍，不测距 |

公式（超声波）：`距离 = 声速 × 时间/2`，声速 `≈ 331.4 + 0.6·T(°C)` m/s，常温取 340 m/s，即 `cm = echo_us / 58`。

选型：**精确定位/到边/防撞优先 VL53L0X/L1X**（IIC，多个用 XSHUT 改地址挂同一总线）；**只要粗略测距/省钱用 HC-SR04**；红外只做触发式避障。VL53L1X 量程到 4m、50Hz，适合需要稍远距离的题目。

## 五、里程计：编码器积分与航向融合

- 编码器（光电正交 AB 相 / 霍尔），用定时器 **正交编码模式（QEI/Encoder Interface）** 自动计数（STM32 TIM 编码器模式 / MSPM0 用定时器捕获或 TIMG QEI 配置）。
- 速度（用于速度环 PID）：

$$ v = \frac{\Delta count}{ppr \cdot 4} \cdot \pi D \cdot \frac{1}{\Delta t} $$

其中 `ppr`=编码器线数、×4 为正交四倍频、`D`=轮径。

- 位移（定距离走 N 米）：对计数累加 `s = Σ Δcount /(ppr·4) · πD`。
- **航向角融合**：纯里程计转弯会累积误差，标准做法是 **里程计算位移 + IMU(yaw) 算朝向**，平面运动模型只对 yaw 积分：

```c
// 差速里程计 + IMU航向
float ds = (ds_left + ds_right) / 2.0f;     // 平均线位移
float yaw = imu_yaw;                          // 用IMU航向替代轮速差积分(更准)
x += ds * cosf(yaw);
y += ds * sinf(yaw);
```

要点：轮速差算出的航向漂移快，**优先用 IMU yaw**；编码器只管「走了多远」，IMU 只管「朝哪」，两者解耦最稳。

## 六、与 K230(CanMV) 通信

K230 板载 USB 转串口（CH342），与主控用 **UART** 交换偏差。建议定长数据帧：`帧头0xAA 0x55 + 类型 + 偏差(int16) + 校验 + 帧尾`，主控 DMA 空闲中断收包解析。

```python
# K230 CanMV 端：回传巡线中线偏差
from machine import UART, FPIOA
fpioa = FPIOA(); fpioa.set_function(11, FPIOA.UART2_TXD)
uart = UART(UART.UART2, baudrate=115200)
err = cx - img.width()//2          # 视觉中线偏差
uart.write(bytes([0xAA,0x55, err & 0xFF, (err>>8)&0xFF, 0x0D,0x0A]))
```

融合策略：**直道/弱光用视觉中线，近距精细对位用灰度/电磁，IMU 兜底航向**。三者按场景加权或状态机切换，避免单源失效甩车。

## 选型快速结论（性价比/精度/成本）

- 循迹：入门 TCRT5000 自拼 → 进阶 **感为 8 路模拟灰度** → 省算力 **IIC 智能灰度**；电磁题用 **10mH+6.8nF LC + 运放 + BAT54S 峰值检波 + 差比和**。
- IMU：练手 MPU6050 → 主力 **ICM-42688-P(SPI)** 或省事 **JY901S(UART直出角度)**。
- 测距：精准 **VL53L0X/L1X(IIC)**，省钱 HC-SR04。
- 里程计：编码器 QEI 测位移 + **IMU yaw 测航向**，解耦融合。

## 🔑 关键要点（速记）

- 电磁循迹核心：20kHz参考线 + LC并联谐振选频（实战L=10mH、C=6.8nF，f0≈19.3kHz对准20kHz）+ 运放放大 + BAT54S肖特基峰值检波 + 差比和(V_L−V_R)/(V_L+V_R)求偏差，换赛道必须重标vmax归一化。
- 光学循迹优选感为8路模拟灰度或IIC/UART智能灰度（后者内部直出中线偏差，省主控ADC与算力）；中线用加权质心法 pos=Σi·xi/Σxi，全丢线要保持上一帧防甩车。
- IMU首选：软件主攻图省事用JY901S（UART直出roll/pitch/yaw，静态0.05°动态0.1°，9轴磁补解决航向漂移）；要高动态自解算用ICM-42688-P（SPI 24MHz、噪声最低、稳定性优于MPU6050/9250/20602）；练手用MPU6050。
- 姿态解算性价比排序：一阶/二阶互补滤波(α≈0.98)最易上线 → Mahony/Madgwick四元数 → 卡尔曼；MPU6050 DMP调库即用但≤200Hz且精度一般。
- 走直线/定角转弯用IMU的yaw做航向反馈；纯6轴yaw会缓慢漂移，转弯靠相对角度增量即可，长时间航向需9轴磁力计或JY901S。
- 测距选型：精准定位/防撞用VL53L0X(2m,±3%,IIC)或VL53L1X(4m,50Hz,IIC，多个用XSHUT改地址挂同总线)；省钱粗测用HC-SR04（cm=echo_us/58，波束宽易误测）；红外只做触发避障。
- 里程计解耦融合最稳：编码器用定时器正交QEI模式测位移（v=Δcount/(ppr·4)·πD/Δt），航向用IMU yaw，x+=ds·cos(yaw)、y+=ds·sin(yaw)；不要用轮速差积分航向（漂移快）。
- MSPM0G3507用12-bit SAR ADC（17路多路复用）配多通道顺序重复转换+DMA一次扫完所有灰度/电感通道，避免逐路阻塞采样。
- K230(CanMV)用UART回传视觉中线偏差，建议定长数据帧(帧头0xAA55+偏差int16+帧尾)，主控DMA空闲中断收包；多源融合按场景加权/状态机切换（视觉走直道、灰度电磁精细对位、IMU兜底航向）。
- 所有光学/电磁传感器对环境（光照、离地高度、铜线电流）极敏感，比赛前必须现场重新标定阈值与归一化最大值，这是循迹稳定的第一前提。
